# Описание

класс WORKER на языке Java, моделирующий сущность «работник» для организации (на примере Университета «Синергия»).
Класс реализован в отдельном модуле и используется вместе с основной программой, которая выполняет ввод данных с клавиатуры, вычисление стажа и вывод фамилий сотрудников, чей стаж превышает заданное значение.

## Запуск программы

для запуска программы необходимо:

1.  скомпилировать файлы командой

```
  bash

  javac Main.java Worker.java

```

2.  запустить проект командой

```
  bash

  java Main

```

## Запуск тестов

для запуска тестов необходимо

1.  скачать "junit-platform-console-standalone-1.10.5.jar" и положить в корень проекта.
2.  скомпилировать файлы командой

```
  bash

  javac -cp "./junit-platform-console-standalone-1.10.5.jar" \*.java

```

3. запустить тесты командой

```
  bash

  java -jar junit-platform-console-standalone-1.10.5.jar --class-path . --scan-classpath
```
